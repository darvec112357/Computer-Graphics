Part 3: Photon Mapping

Files modified are arealight.cpp and photonmapper.cpp. This task took about 3 days.

Implementation

For the preprocess() function, use 2 nested while loops. The outer loop counts the total number of emitted photons while the inner loop trace each emitted photon. Once the photon hit a diffuse surface, store it. When there is enough stored photons, break the loops. We need another variable em_count to count the total number of emitted photons, which will then be used in Li() for density estimation. In addition, using random emitters might be a better way than using for-loops because using for-loop will only sample photons from the last emitter. The Li() function is similar to path_mis.cpp except that this time we have to take into account photon density.
